## 💻 Run the Project

<br>

Clone the project

```bash
  git clone https://github.com/Caioscg/Projeto-ES-api
```

Navigate to the project directory

```bash
  cd Projeto-ES-api
```

Install the dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
